# ImageSearcher
基于LIRE搭建的图像检索

### 服务端 
Java编写

### 客户端 
Qt编写

### 通信方式 
Tcp socket通信

### 测试图像库  
Corel-1000图像库<br>
包含10类，每类100幅图像（印度、海滩、建筑、大巴、恐龙、鲜花、骏马、大象、山河、食物）
